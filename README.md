

# 📖 BookGPT — Анализ текстов с Yandex GPT

**BookGPT** — это веб-приложение для интеллектуального анализа текстов с использованием нейросетей Yandex GPT. Сервис позволяет загружать книги, получать краткое содержание, задавать вопросы по тексту и искать фрагменты с помощью AI-ассистента.

## ✨ Основные возможности

| Возможность | Описание |
| :--- | :--- |
| 📁 **Загрузка текстов** | Ручной ввод, загрузка `.txt` файла или тестовая книга для демонстрации. |
| 🔍 **Определение кодировки** | Автоматическое определение кодировки (UTF-8, Windows-1251). |
| ✂️ **Умная разбивка** | Большие тексты разбиваются на смысловые чанки для оптимальной обработки. |
| 📝 **Краткое содержание** | Генерация саммари книги с помощью YandexGPT. |
| 💬 **Чат с ассистентом** | Задавайте вопросы по контексту загруженного текста. |
| 🎯 **Два режима** | "Вопрос" (ответ AI) и "Поиск" (поиск цитат по тексту). |
| 🌐 **Интернет** | Опциональное подключение интернета для расширенных ответов (в разработке). |
| 📱 **Адаптивность** | Корректное отображение на всех типах устройств. |

## 📂 Как загрузить тексты

1.  **Ручной ввод:** Вставьте текст в поле на главной странице и нажмите *"Анализировать"*.
2.  **Загрузка файла:** Нажмите на область загрузки или кликните по иконке облака, выберите файл `.txt` (до 10 МБ).
3.  **Тестовая книга (католог):** Нажмите кнопку *"Загрузить тестовую книгу"* для быстрой загрузки уже существующего текста.

## 💡 Примеры работы

Ниже приведены примеры взаимодействия с сервисом на основе произведения И.С. Тургенева *«Муму»*.

### 1. Генерация краткого содержания
После загрузки текста и нажатия *"Анализировать"*, AI генерирует саммари:
> В Москве в сером доме с белыми колоннами жила барыня-вдова. Среди её прислуги был глухонемой дворник Герасим — мужчина богатырского телосложения, который усердно исполнял свои обязанности... (сокращено).

### 2. Режим "Вопрос"
**Вопрос:** `Кто такая Муму?`

**Ответ:**
> Муму — это собака, которая живёт у Герасима.
> *Цитата:* «Перед окном был разбит палисадник, и на самой средней клумбе, под розовым кусточком, лежала Муму и тщательно грызла кость» (ЧАСТЬ 5).

### 3. Режим "Поиск"
**Запрос:** `Степан заметил, что он её утопил.`

**Результат:**
> *Краткое описание:* Степан выражает уверенность в том, что Герасим утопит собаку.
> *Цитата:* «Ну, да, — заметил Степан, — он её утопит. Уж можно быть спокойным. Коли он что обещал...»

### 4. Вопрос вне контекста
**Вопрос:** `Кем был Печорин?`

**Ответ:**
> В тексте нет информации об этом.

---

> **⚠️ Примечание:** Функция поиска с использованием интернета находится в стадии разработки и может работать некорректно.
